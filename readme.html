初二七班班级抽奖系统

一个基于Web的班级抽奖系统，包含奖励和惩罚两个奖池，支持奖品记录管理。界面简洁美观，操作简单直观。

功能介绍

奖池系统

A奖池（奖励）

  • 免一次作业
  • 免一周值日
  • 换座位一周
  • 获得零食

B奖池（惩罚）

  • 给全班带礼物
  • 一张试卷
  • 英语作业x2
  • 罚站两节课

核心特性

  • 每个奖池10张卡片，随机分配奖品
  • 精美的翻卡动画效果
  • 奖品记录管理
  • 奖品使用确认系统
  • 数据本地存储
  • 响应式设计

快速开始

安装

# 克隆项目
git clone [项目地址]

# 进入项目目录
cd [项目名称]

# 安装依赖
npm install

# 安装开发依赖
npm install fs-extra --save-dev
npm install nodemon --save-dev

运行

# 预览模式 (端口4000)
npm run dev

# 构建
npm run build

使用指南

抽奖流程

  1. 选择奖池（A/B）
  2. 输入学生姓名
  3. 点击开始抽奖
  4. 选择任意卡片
  5. 查看抽奖结果
  6. 其他卡片自动展示

奖品管理

  1. 点击"抽奖记录"按钮
  2. 查看所有学生的奖品记录
  3. 点击"使用"按钮消耗奖品
  4. 确认使用操作

项目结构

├── public/          # 静态资源
│   ├── css/        # 样式文件
│   ├── js/         # JavaScript文件
│   └── index.html  # 主页面
├── server.js       # 服务器入口
├── build.js        # 构建脚本
└── package.json    # 项目配置

技术栈

  • 前端：HTML5、CSS3、JavaScript
  • 后端：Node.js、Express
  • 存储：localStorage

开发环境

  • Node.js >= 14.0.0
  • npm >= 6.0.0

注意事项

  • 数据存储在浏览器本地
  • 清除浏览器数据会导致记录丢失
  • 建议定期备份数据
